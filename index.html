Introdução ao Javascript

Criar repositório no GITHUB.

Abrir GIT BASH e digitar: git clone "endereço do repositório do GitHUB", e clicar em ok.

Abrir Visual studio Code

Open in file "Open Folder..." or crt+k ctrl+o e selecionar a pasta do repositório local clonado do repositório remoto.

No VS criar aba Index.html e digitar o código.
